Cell Modification > Flip

The flip command is used to flip a cell to a value. The syntax for this command is "f" followed by the cell id that you would like to flip (0-8). This unlike the set command flips the value with uncertainy of a cells value.

For example if you wanted to flip cell 2 you would do "f2". If you wanted to flip cell 8 you would do "f4".
