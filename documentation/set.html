Cell Modification > Set

The set command is used to set a cell to a value. The syntax for this command is "s" followed by the cell id and then the value you want to set it to. This unlike the flip command sets the value for certainy of a cells value.

For example if you wanted to set cell 0 to 1 you would do "s01". If you wanted to set cell 8 to 0 you would do "s80".
